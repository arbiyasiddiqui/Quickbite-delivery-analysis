# QuickBite Delivery – Performance Analysis Dashboard

## Project Overview
This project analyzes delivery performance and customer experience data for a food delivery application called QuickBite.

## Business Problem
The company faced operational challenges such as delivery delays and inconsistent delivery partner performance across different cities.

## Tools Used
- Power BI
- Google Sheets
- Data Visualization

## Key Metrics Analyzed
- Total Orders
- Delivery Delay
- Customer Ratings
- Payment Modes
- Partner Performance

## Insights
- Identified cities with higher delivery delays
- Evaluated delivery partner performance
- Analyzed customer rating distribution
- Provided insights to improve delivery efficiency and customer satisfaction
